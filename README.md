# digital-literacy-project

## Project Overview

The Digital Literacy project is part of our Digital Literacy course. It helps students develop digital skills needed today. We cover awareness, online presence, coding, communication and cyber safety.

This project has five tasks. Each task teaches us about tools and how to behave online.

## Task 1: Digital Literacy Infographic

I made an infographic using Canva. It teaches people about literacy. The infographic covers:

* What is digital literacy?

* Tools for students

* Safe internet use

* Email etiquette

* Online presence

Folder: `task-1-presentation/`

## Task 2: Digital Portfolio

I created and updated my profiles on GitHub, LinkedIn and stackoverflow. These platforms help build an online presence. I learned how to showcase skills. Connect with others.

Folder: `task-2-portfolio/`

## Task 3: Coding & Collaboration Platforms

I used coding and collaboration tools. I solved a beginner problem on HackerRank. This improved my coding skills. I also made a Google Form quiz on literacy. This helped me learn how to collect and analyze responses.
